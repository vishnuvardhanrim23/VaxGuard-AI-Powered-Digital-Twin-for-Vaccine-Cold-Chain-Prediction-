# VaxGuard Final Streamlit Dashboard

## What this app includes

- Executive Summary
- Active Shipment Monitoring
- Temperature & Humidity Trends
- Risk Prediction Dashboard
- Remaining Safe Time Prediction
- Vaccine Loss Analysis
- Financial Impact Analysis
- Geographical Shipment Tracking
- Recovery Recommendation Panel
- Digital Twin Simulation
- Model Performance Dashboard
- Dataset Page

## Dataset

The app generates a clean synthetic dataset:
- 7 vaccine types
- 200 records for each vaccine
- 1400 total records
- Each shipment has 200 vaccine units

## How to run

Open CMD inside this folder and run:


pip install -r requirements.txt
streamlit run app.py
